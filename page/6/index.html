【周记】25 被社交媒体支配中

发表于 2022-06-26 分类于 周记 阅读次数：

过去的一周感觉自己的可支配时间被压缩的厉害，也许最近依旧沉迷羽毛球，基因中狩猎的本能让我努力的从身边汲取各种羽毛球相关的信息，每个话题或者物件都耗费了我大量的时间来无意识的浏览，搭配个性化算法推荐的那些”有趣“的文章和视频，犹如泥沼令人无法自拔。
这礼拜的一些关于羽毛球的关键词：
• 首先是种草了Bauerfeind护膝，一款价值不菲但身边口碑很好的护膝，由于右膝盖半月板一直有伤，最近打球又比较频繁，感觉膝盖有些吃不消。亲测护膝质量的确不错，不容易掉而且透气和支撑都比较好。
• 然后又从朋友手里用一把vtzf2lcw换了一把2010年的Arcsaber z Slash Taufik（azth）。ArcZ是继08年弓10之后YY推出了两款特别弓剑中的一把，其中ArcFB主打轻量，ArcZ主打小拍面进攻，陶肥也短暂使用过。这把是3UG4，去底胶之后手感还不错，成色也很新，应该打的机会不多，目前主要还是打3U弓10（BG80）和弓11（BGAS）。
• 跟教练练球。这礼拜捡漏约到了湾区这边一个有名的前国手教练，上了一堂两小时的教练课，课程容量还是蛮大的，三个人一组半块场地一直练，由于之前已经连续打了三天，体力有些跟不上，学到了很多实用的技术和知识，包括平抽挡，高远球和杀球的动作，击球后的脚步等等。希望有机会能够一直跟着教练练球。

【周记】24 太浩湖遛狗

发表于 2022-06-19 更新于 2022-06-26 分类于 周记 阅读次数：

这周有点摆烂，新的sprint的第一周，主要的工作是做一些POC，没有太多代码的任务。于是每天白天悠哉的看看文献教程，晚上打球。（后来证明低效的第一周会给后面带来很多无形的压力）。
本周给咖啡机做了一次清洁，周五出门前又做了一次保洁。

星期五下午约上另外三家哈士奇家长一起出发前往太浩湖周末小聚。雪季后我们俩便没有再去过太浩湖，这礼拜正值湾区气温飙高，看天气预报太浩湖那边之后10度上下，十分期待。
星期五的下午，湾区堵得水泄不通，平时4小时上下的路程，我们开了5个半小时。堵车的罪魁祸首是车祸，不知为啥高峰时期车祸就出现的尤为频繁。
自打从科州回来，感觉自己对开长途的阈值又提高了不少，现在一趟Tahoe两百多mile都不需要停车休息了，一口气就沿着I80开到185出口，然后南下来到湖边小木屋。

这是四条哈哈第一次一起出去旅行，也是八位家长第一次同行。来到小木屋后的四条狗子开心的不得了，一直疯玩到吃过晚饭，就渐渐累了，各自找地方趴着。家长们意犹未尽，就着博文带的蓝方，玩了会一夜狼人，Dixit，我和JY对动脑筋的游戏不是很擅长，不过也和大家玩的很开心。晚上11点，印尼公开赛进入到半决赛，看到王祉怡力克何冰娇，挺进决赛 （但最终不敌小戴，拿下超级1000亚军）。

接下来的两天，我们冒着六月飞雪去湖边hiking，也在暖阳下去湖中划船，Oyuki虽然怕在泳池里游泳，但在船上比较放松，好奇的四处张望，还能在浅水里拉着我们的kayak到处跑。

愉快的一周的最后几个小时，我和JY开着车，带着Oyuki，追着夕阳从太浩湖回到温暖的家。

【周记】23 健康为上

发表于 2022-06-12 更新于 2022-06-26 分类于 周记 阅读次数：

这一周仿佛按下了加速键，前三天JY出差，我便每天在家工作，三餐，喂狗，打球，一晃就过去了。
这周验血做了一次复查，距离去年年底检查出一系列问题过去整半年，好消息是绝大多数指标如今已经恢复正常。这主要归功于生活习惯的180度转变——提高运动频率，控制饮食，戒酒，所有这些帮助我成功减重7kg，回归正常体重，同时规律的运动有助于帮我保持较好的心肺功能和竞技状态。具体来看，代表最近三个月平均血糖水平的糖化血红蛋白（Hemoglobin A1C），总胆固醇和尿酸都恢复到了正常水平，然而持续了三个月的血糖监测表示我日间血糖值依旧偏高，三餐依旧要控制淀粉和糖分得摄入。
这周末带着狗子参加哈士奇的聚会，又来到Fort Funston，其中Lucky和Saber都是第一次来FF，狗子们在一起又度过了一段快乐的时光。回来之后O妈倾情剪辑的视频在小红书上大获好评，收割一众姨母笑。
周末则邀请好友来家里烤串，灼灼夏日，和朋友在小院里撸串喝啤酒吃西瓜，谈天说地，真是畅快。

【周记】22 运动不止

发表于 2022-06-05 更新于 2022-06-26 分类于 周记 阅读次数：

六月的第一周燥热而又平淡。
这礼拜狗子的脖子彻底好了，第一天是Memorial Day放假，在costco买了些上好的牛排在家煎着吃。
这周约到了一个按摩店的问诊，检查发现自己的颈椎变得跟尺子一样笔直笔直，于是听从医生的推荐搞了一个Posture Pump每天在家弯曲颈椎。同时还约了每周两次按摩+电疗+正骨+激光来治疗自己的网球肘和肩袖。
这周末JY去西雅图出差三天，这是2020年2月从温哥华回来之后，她第一次坐飞机，虽然身边着实有不少人在疫情期间飞来飞去，但我们一直保持自驾长途，因此多少还是有些紧张，出发那天她全程戴N95+护目镜，并表示机场和飞机上都有人不戴口罩，公司中戴口罩的更是寥寥无几。相比之下我们公司这方面更保守一些，不但不对进一步RTO做严格要求，同时恢复公司内公共场所戴口罩的要求。近期美国疫情卷土重来，单日确诊突破10万，身边陆续有朋友中招，大有一波带走的节奏，还是希望自己不要感染，不想让三年来的保护措施功亏一篑。
这礼拜打了不少球，尤其是周末和一对老搭档搞了几句男双之后，被各种教做人，暴露的问题主要围绕着男双走位和节奏，一个是网前球接完不要回动，另一个是网前不要抢过顶的球，充分相信后场的队友。另外由于节奏略快于自己平时习惯的节奏，因此经常性的等球，吃假动作，缺少启动步，导致被动。

刷到这个双打三字经，每次打球前默念三遍，最直观的变化是之后打球的时候，能够更积极的争抢网前机会，逼迫对手起球。

【周记】21 五月再见

发表于 2022-05-30 更新于 2022-06-26 分类于 周记 阅读次数：

五月的最后一周，迎来Memorial Day长周末。
本周顺利拿到了变态的Apple Fitness 五月挑战（32300卡路里），为了实现这个目标，我基本上每周要打5次球，每次羽毛球平均消耗1500卡，网球800卡。持续打球使自己的身体多少有些吃不消，最近手肘，膝盖，脚踝和肩膀都有些疲劳，我拿来了JY的加热带来给手肘热敷，感觉很有效，同时还买了一台TENS（经皮神经电刺激），每天给自己电上半小时，感觉还不错。
狗子这礼拜出现了些状况，简单形容来说每当他躺久了翻身起来的时候就会惨叫一声，类似被踩到脚一样歇斯底里的叫，但叫过一声之后就没事了。我和她妈都很着急，又是按压检查，又是网上搜索。神奇的是这家伙无论我按哪里他都不疼不叫，胳膊腿都检查了一圈找不到痛点，网上查了查感觉像是神经炎或者脊柱/腰出了问题。与此同时我还注意到他不敢大力的甩头，终于把问题定位在脖子上。约了Pet Friends要下周才能去看，于是又约了Sage 急诊，医生和他妈也一波检查没有查出啥问题，应该是脖子往左转的时候疼导致的，于是验了血，开了一瓶止疼药就回家了。回家之后便没事了，此后也再也没叫过，我们总结来看可能是落枕了。
这周末去了Redwood City的Stulsaft park，这里是一个有山有水的狗公园，环境很好，全程可以offleash，还可以顺便hiking。继Fort Funston之后又发现了一处狗子的快乐乐园。Oyuki全程无比开心，上山下水，体验了一回做「野狗」的快乐。
周日晚上第一次给羽毛球拍拉线，和网球拍相比，羽球拍的线孔模式比较统一的72孔，我这次学习的是标准的Yonex两线四结拉线法，横线4.5m，竖线5.5m，四个结都用的结束结。横线27，竖线27.5，但实际估计只拉到了25-27。需要注意的是从网球拍设置调到羽毛球拍设置，不仅要调配重，拍夹，还有飞夹的力度，总之变化很多，每次换的时候要小心调整好之后再开始拉线。这次又拉了3个小时，希望下次应该会有很多的进步。
长周末在家刷完了「爱死机」第三季和「Stranger Things」第四季。这季故事交代了很多011和大魔王的身世之谜，但结局远没有解决当前各个场景的问题，期待「致命钥匙」的更新。

这礼拜在家继续写Javascript，由于跟我一个屋的同事成了小阳人，周四也没有去公司。这礼拜在家做了好几顿饭。这周还开始坚持每天早上一杯蛋白粉，感觉对于控制血糖有一定的帮助。
羽毛球馆的会员团购顺利完成，从5/11开始生效，这周用了两次，以后周三的东湾活动就转移到周二的Bintang了。
本周汤尤杯进入淘汰赛环节，并于周五迎来决赛。中国女队挺近决赛，但不敌韩国女队，男队拉胯的很，3-0 不敌印尼，止步四分之一决赛。
从阵容上来看，今年男队派出了更多年轻球员，但正如林丹赛后的评论，年轻不是输球的借口，中国队还是背负了太多的压力和期望，我们从比赛中没有看到如最后夺冠的印度队年轻球员眼中的拼劲和杀气，整体感觉略显沉闷。女队相比起来整体实力还是要高出很多，打的也很顽强，决赛我看了一单一双三单三场，雨菲大帝第一轮展现出强大的心脏，决胜局20-17都给她追回来了，我作为观众都感到压力山大。这场比赛也让我记住了韩国女单安洗莹的名字，打的同样顽强而出色。但同样暴露出了一些问题，我觉得队员们场上的状态起伏还是比较大，经常能看到领先5分多，然后又被追回的情况，凡尘组合的输球显得有些轻敌，而新科亚锦赛女单冠军的王祉怡显然在决胜局中心态崩塌，被21-8击垮。
这周末的好消息是吉林逐渐解封，爸妈也终于可以回吉林老家和家人团聚。但上海的封城政策依旧严格执行，人们在“维持动态清零政策不动摇”的大运动下，小声喧哗。
这周体重有所反弹，最近大鱼大肉吃的有点凶，连着做了好几天大米饭，未来的一个礼拜要加强自我管理。
本月Apple Watch challenge完成 20487/34300 卡路里。
