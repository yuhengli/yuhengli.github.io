【周记】06 冬奥情结

发表于 2022-02-13 更新于 2022-02-19 分类于 周记 阅读次数：

二月的第二周，褪去过年的热闹气氛，在家平静度过。

我们将圣诞的装饰和9ft高的圣诞树统统收纳到四个大箱子，关掉了房前屋后的圣诞彩灯，房子虽然瞬间变得冷清了不少，但同时也变得简单而温馨。

这礼拜coco爸妈要去阿拉斯加玩，便把coco寄养在我家一周。开启愉快的二狗生活。拉布拉多的性情和习惯和哈士奇大相径庭，两只狗虽然疯起来招数都差不多，但显然拉布对主人更加热情，懂事听话但也更加敏感，而哈士奇则更加独立，我行我素（有食物除外），一副乐天派。其实两只大狗白天都可以乖乖的自己和自己玩，或者呼呼大睡，家里有一只或是两只最大的差别并不在于需要耗费更多的精力来陪他们玩，而是家里产生了双倍的狗毛。平时扫地机器人一趟下来可以集满一盒子狗毛，自从coco来我家，扫地机器人的尘盒没等工作完就堵了，我也需要每天额外手动来吸一遍地。

本周的另一个关键词是冬奥会。上周末大幕拉开之后，每天晚上我们都会在家认真看比赛。我比较感兴趣的项目有单板滑雪和自由式滑雪的Big Air，slope style，还有短道速滑。苏翊鸣，谷爱凌的比赛都从预赛看到决赛，两位小将的冬奥首秀都收获满满，真的为他们高兴。短道速滑虽然失掉了男团5000米接力着实可惜，但在混合接力上有所突破，后面的单人比赛希望能够发挥出好的水平。

【格物】学习给木头上漆

发表于 2020-08-04 分类于 格物 阅读次数：

新买的桌面只有最简单的hard wax，摸起来感觉不是很有质感，而且walnut veneer也需要额外的保护，因此打算给它封印一层清漆。

目前比较主流的保护漆包括Poly和Lacquer，每种漆的溶剂又分水性还是油性，上漆方式可以是刷，涂或者喷。

TLDR

我最后选择的是 Satin Polyurethane Wipe-On 主要特征如下

• Poly

• Wipe-on

• 油性

• 缎面 Satine

选择这款漆的原因主要是操作简单，漆面比Lacquer更耐用，Satine缎面的反光柔和，不会有High Gloss的塑料感，更突出Walnut本身的质感。
