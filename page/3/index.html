【杂谈】这个夏天，选一条柏香籽

发表于 2022-08-07 更新于 2022-08-08 分类于 杂谈 阅读次数：

最近关注了很多柏香籽的商家，也在各方渠道读了很多关于柏香籽的文章，其实所谓各方渠道，主要是小红书，Youtube上只有寥寥几篇文玩博主的综述性的视频，Google上也搜不到什么，显然这东西还没有渗透进简中之外的世界。我没有安装抖音和B站，想必那里的信息源和小红书也差不多。
小红书真可谓种草神器，只要你想关注的话题，上面都有若干篇质量还算不错的文章帮你测评，试玩，总结，避雷… 真是一刷就停不下来。里面的若干篇文章也煞有介事的帮你厘清当今市面上柏香籽商家的乱象，并”真诚“的推荐了几个工作室。
比较两个极端的例子，其中最大的一家店「WC」基本上打通了从上游供货到下游营销的一整套商业链，淘宝店，直播都做的有声有色，款式充足，全品类覆盖，小几百到大几千的籽都能买得到，推广做的也很到位，甚至在网上发起所谓的「千图计划」，通过顾客回图，既宣示了自己的品牌影响力，也给潜在客户提供了很多设计灵感，大大的种了一波草。而除了寥寥即家头部的大卖家，我所了解到的更多的是小工作室，他们或是从批发商处拿籽，或是小规模从本地收籽，联系加工厂或自己手工加工，然后通过小红书，微信等平台小规模贩售。

【周记】25 被社交媒体支配中

发表于 2022-06-26 分类于 周记 阅读次数：

过去的一周感觉自己的可支配时间被压缩的厉害，也许最近依旧沉迷羽毛球，基因中狩猎的本能让我努力的从身边汲取各种羽毛球相关的信息，每个话题或者物件都耗费了我大量的时间来无意识的浏览，搭配个性化算法推荐的那些”有趣“的文章和视频，犹如泥沼令人无法自拔。
这礼拜的一些关于羽毛球的关键词：
• 首先是种草了Bauerfeind护膝，一款价值不菲但身边口碑很好的护膝，由于右膝盖半月板一直有伤，最近打球又比较频繁，感觉膝盖有些吃不消。亲测护膝质量的确不错，不容易掉而且透气和支撑都比较好。
• 然后又从朋友手里用一把vtzf2lcw换了一把2010年的Arcsaber z Slash Taufik（azth）。ArcZ是继08年弓10之后YY推出了两款特别弓剑中的一把，其中ArcFB主打轻量，ArcZ主打小拍面进攻，陶肥也短暂使用过。这把是3UG4，去底胶之后手感还不错，成色也很新，应该打的机会不多，目前主要还是打3U弓10（BG80）和弓11（BGAS）。
• 跟教练练球。这礼拜捡漏约到了湾区这边一个有名的前国手教练，上了一堂两小时的教练课，课程容量还是蛮大的，三个人一组半块场地一直练，由于之前已经连续打了三天，体力有些跟不上，学到了很多实用的技术和知识，包括平抽挡，高远球和杀球的动作，击球后的脚步等等。希望有机会能够一直跟着教练练球。

【周记】24 太浩湖遛狗

发表于 2022-06-19 更新于 2022-06-26 分类于 周记 阅读次数：

这周有点摆烂，新的sprint的第一周，主要的工作是做一些POC，没有太多代码的任务。于是每天白天悠哉的看看文献教程，晚上打球。（后来证明低效的第一周会给后面带来很多无形的压力）。
本周给咖啡机做了一次清洁，周五出门前又做了一次保洁。

星期五下午约上另外三家哈士奇家长一起出发前往太浩湖周末小聚。雪季后我们俩便没有再去过太浩湖，这礼拜正值湾区气温飙高，看天气预报太浩湖那边之后10度上下，十分期待。
星期五的下午，湾区堵得水泄不通，平时4小时上下的路程，我们开了5个半小时。堵车的罪魁祸首是车祸，不知为啥高峰时期车祸就出现的尤为频繁。
自打从科州回来，感觉自己对开长途的阈值又提高了不少，现在一趟Tahoe两百多mile都不需要停车休息了，一口气就沿着I80开到185出口，然后南下来到湖边小木屋。

这是四条哈哈第一次一起出去旅行，也是八位家长第一次同行。来到小木屋后的四条狗子开心的不得了，一直疯玩到吃过晚饭，就渐渐累了，各自找地方趴着。家长们意犹未尽，就着博文带的蓝方，玩了会一夜狼人，Dixit，我和JY对动脑筋的游戏不是很擅长，不过也和大家玩的很开心。晚上11点，印尼公开赛进入到半决赛，看到王祉怡力克何冰娇，挺进决赛 （但最终不敌小戴，拿下超级1000亚军）。

接下来的两天，我们冒着六月飞雪去湖边hiking，也在暖阳下去湖中划船，Oyuki虽然怕在泳池里游泳，但在船上比较放松，好奇的四处张望，还能在浅水里拉着我们的kayak到处跑。

愉快的一周的最后几个小时，我和JY开着车，带着Oyuki，追着夕阳从太浩湖回到温暖的家。

【周记】23 健康为上

发表于 2022-06-12 更新于 2022-06-26 分类于 周记 阅读次数：

这一周仿佛按下了加速键，前三天JY出差，我便每天在家工作，三餐，喂狗，打球，一晃就过去了。
这周验血做了一次复查，距离去年年底检查出一系列问题过去整半年，好消息是绝大多数指标如今已经恢复正常。这主要归功于生活习惯的180度转变——提高运动频率，控制饮食，戒酒，所有这些帮助我成功减重7kg，回归正常体重，同时规律的运动有助于帮我保持较好的心肺功能和竞技状态。具体来看，代表最近三个月平均血糖水平的糖化血红蛋白（Hemoglobin A1C），总胆固醇和尿酸都恢复到了正常水平，然而持续了三个月的血糖监测表示我日间血糖值依旧偏高，三餐依旧要控制淀粉和糖分得摄入。
这周末带着狗子参加哈士奇的聚会，又来到Fort Funston，其中Lucky和Saber都是第一次来FF，狗子们在一起又度过了一段快乐的时光。回来之后O妈倾情剪辑的视频在小红书上大获好评，收割一众姨母笑。
周末则邀请好友来家里烤串，灼灼夏日，和朋友在小院里撸串喝啤酒吃西瓜，谈天说地，真是畅快。
