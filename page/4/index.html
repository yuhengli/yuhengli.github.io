【周记】16 四月之声

发表于 2022-04-24 更新于 2022-06-26 分类于 周记 阅读次数：

过去的一周，体重有所回升，JY说是因为我做好吃的的频率提高了…
的确这周抽空做了很多顿饭，从炸鸡翅到羊蝎子，从炒饭到鱼头泡饼，的确满足了口腹之欲，但同时也少吃了好多顿沙拉，从一月开始持续下降的体重在此遇到瓶颈。
但这礼拜的运动也没有停下，打了三天羽毛球三天网球，网球肘表示亚历山大。周五和Yuki爸打了一回网球，深感对方内功深厚，和高手过招，对自己也有很大的影响，我开始发觉自己平时收羽毛球的影响，手臂挥动过多，而忽视了转体的重要性，尤其是当对面来球沉重的时候，要找到用身体往前压球的感觉，这就要求自己必须保持完整的动力链，而不是单纯地用手臂来击球。另一个提高是发球的质量，得益于周末和Andy的多盘比赛，多打比赛对自己的发球还是有很大的提升的。
这周是回办公室的第二周，周二去办公室set up好了iMac Pro，周四和San Diego office过来的同事亲切会晤，周五大家一起去Top Golf 团建。

【周记】15 回公司上班

发表于 2022-04-15 分类于 周记 阅读次数：

这周过真快。
星期二是时隔两年后重新「回到」公司上班，之所以用引号是因为两年前我没有从这里离开过（虽然此时此刻我还用的老东家的电脑来敲这篇文章）。我不得不承认时隔两年，我对于在公司中，与人面对面的交流既期待又陌生，仿佛是打开了一台尘封了两年的老电脑，每一个操作都伴随着一段记忆的唤醒。由于一大早有个面试，所以我提前一天早早地订好了会议室，九点就来到公司，Infinite loop的办公室是两个人一间，我们层没有小会议室，我定了一个6人间，面试结束之后便看到了很多同事，大家见面的感觉和见网友无异。随后还带着同是疫情后加入的朋友一起扫楼。中午老板带我们出去吃的饭，因为公司餐厅已经人山人海，到处都排着长队。沉寂了两年的公司仿佛一下子被点燃了。人们相互问候，就像暑假后返校的学生一样。下午的Debrief一整组的人头一次围坐在一张会议桌前，面对面开会的感觉很奇妙，我可以盯着对面同事的肚子听他讲话。晚上下班后我们又去逛了一圈Apple Store，之后带外卖回家。

这周湾区开始降温下雨，取消了周二的网球课，直到周五才出门打了一场，不过周三的羽毛球打的很爽，被一对混双虐了两个小时，只拿下一局。总结来看，在节奏提高的前提下，自己的失误明显增多，回球路线变的单一，进攻也没有杀伤力，无球跑位有时候也会出问题，接下来的训练要有针对性的全面提高技术。

依旧持续关注着国内疫情，长春封城近两个月，上海也封了半个月了，各种奇葩和突破底线的故事屡有发生，这里就不一一列举了。
这周基本把卡农拿下了，接下来起了几个头，不知道要先练「summer」还是「湘伦小雨四手联弹」亦或是些其他的什么。

进入四月，过去的一周略显平淡。每天除了工作，运动，剩下的很大一部分时间用来关注日益魔幻的国内疫情。每天都不断有突破底线的故事发生。相比东北人民的隐忍和自娱自乐，上海人民的呼声明显要高抗许多，也就能在各个平台听到更多的冲突与悲剧。

【周记】13 魔幻抗疫

发表于 2022-04-02 分类于 周记 阅读次数：

本周是三月的最后一周，我在这个星期六的下午，一鼓作气，第一次把写周记的任务在本周内完成，过久的回忆让文字失去新鲜度。

这周长春的疫情依旧没有好转，同时中国其他城市，诸如上海，济南也陆续见到阳性病例。周五听说三亚也出现了5例，准备（或者已经）封城了。眼看爸妈所在的城市也将收到波及，心里还是挺着急的。过去的一周，在疫情和封城的笼罩下，每天都在被各种魔幻的剧情刷屏。从一开始的吉林农业科技学院爆发聚集性疫情，应对无力，最后导致6556学生被转运到临近城市，分散隔离。到后来由于拖欠修建方舱的农民工工资而导致大打出手。同时，更多的老人、病人吃饭，就医难的问题被屡屡曝光，大家开始「晒晒我家蔬菜包」活动，后来又听说这个活动的政府官方发起者被免除职务，党内警告处分，至此活动宣告翻车。再到最近某小区微信群传出的主人「馋嘴」吃掉自家宠物狗， 上海婴幼儿隔离点安排200个孩子只有10名护士照顾，吉林一病人因无法得到及时救治而上吊自杀，上海120拒绝给路边急需抢救的病人提供AED而导致病人死亡… 尽管官方通过各种途径删帖、辟谣或者致歉，尽管我理性的希望这一切都未曾真实发生过，但时至今日，在上面严格的「清零」政策的高压下，出现这些「个例」我一点也不觉得奇怪。毕竟我的三姨就是一个需要每隔两三天就去医院做透析的病人，听到家里人劝她能多坚持几天就在家挺挺，看到她自拍的小视频，穿着全套防护服艰难步行两个多小时去医院做透析，而她显然既不是第一个，也不是最后一个治疗遭遇不便的病人。、

有一个段子说：“上海现在就是一群疲惫不堪的医护把一堆濒临死亡的病人拦在医院外，守着病房生龙活虎的无症状感染者刷手机。”

我不知道该说些什么。

【周记】10 一周好物

发表于 2022-03-13 更新于 2022-03-16 分类于 周记 阅读次数：

过去的一周波澜不惊，工作上忙着做这一轮的release，其余时间恶补一波特斯拉的相关知识。

自从订车之后，短短两周的时间里迎来了两轮Tesla的涨价，第一次1k，第二次3k，丧心病狂的涨价背后，折射出潜在买家在油价破7时代的躁动不安。看了些国内外关于特斯拉横向和纵向的测评，总结出一些心得。

首先Tesla在税率优惠的政策下，在国内的定价远低于国外竞品，比如宝马i4，奥迪e-tron，而在30万上下这个价位素质表现又碾压同价位电车，可以说是bug级的存在。

其次Tesla的两大利器是制造和创新。马斯克曾表示自动驾驶和电池都可以被复制，而特斯拉的核心竞争力在于生产制造。通过各种技术的创新，特斯拉不断的简化，优化生产制造流程，放眼更大规模的生产，用早期车主的信仰来为未来的平价电动车打基础。这里我仿佛看到了Ford当年生产Model T的场景重现。创新方面，特斯拉将全部精力放在安全和节能上，知乎的「拆车实验室」 出了一期节目，全面的拆解了特斯拉Model Y，印象深刻的是八通阀和4680电池，前者能够自如的在15中模式之间切换，合理调配车内热量，后者则创新性的推出无极耳设计，大大提高了电池的能量密度。此外特斯拉的Giga Press 6000吨级超级压铸机实现了前后车身的一体压铸，减少了焊点和零件，并增强了车身整体强度，大大的优化了车身制造的工艺，同时也提高了制造门槛。

三月一开始先花了两天刷完了「开端」。堪称国产悬疑精品，虽然故事设定没有「白夜追凶」新颖，但小演员演技在线，无限循环中还是有很多人物和情节可以拿来推敲，足够扣人心弦。对于车上每个人物的描写，让我不禁想到东野圭吾的叙事手法，围绕故事主线的每个小人物背后都有属于自己的故事。然而美中不足的是故事结尾太过伟光正。看到十集刚过，我甚至开始脑洞，其实李诗情所在的循环不过是嵌套在刘丹饰演的陶映红和司机二人的大循环之中，他们二人义无反顾的不惜炸掉整辆公交车，其实是为了挽救更大的灾难… 总之刷了十几集，实在舍不得让白白和麦麦一觉睡到天亮😬急速刷完「开端」之后生物钟被调的有些晚了。
