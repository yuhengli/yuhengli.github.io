【周记】31 剪不断

发表于 2022-08-07 更新于 2022-08-08 分类于 周记 阅读次数：

八月第一周，也度过了来美国的第七年，加入Apple的第一年。
这一周我终于结束了无止境的网购，再最后两颗珠子成功抵达转运站之后，我收到了这批转运商品的全家福，并告诉自己，到此为止。就像上一篇杂谈中写的那样，珍惜与他们的缘分，并开始培养与他们的关系，让他们成为不被定义的文玩，也无需再羡慕他人手上的东西，因为无价，所以比较不成立。
这礼拜还尝试戒断小红书，但实际操作起来竟然如此困难，我大概卸掉了三次，又因为各种原因重新装了回来，甚至怀疑自己产生了戒断反应，具体来说大概是首先在空闲时间变得焦躁，急于在手机屏幕上熟悉的位置找到那个图标并进去无意义浏览，其次是当遇到心中的问题，或者听到什么新鲜的事情，渴望去小红书上搜索答案，或者跟进事态发展进度。而一但开始浏览，便停不下来，我参与的话题十分广泛，从柏香籽和文玩，到F1比赛，到八卦，再到最近斯巴鲁被称为「杂牌车」的热点话题。这礼拜开始给新入的松石老型珠泡油，我将对比图发到上面，并渴望看到别人的回复，我又发了一张自己最近打的球拍，同样渴望别人的回复。而当真的收到通知，看到有人评论的时候，心中便舒畅起来。我知道自己「中毒」颇深，希望下礼拜陆陆续续收到的快递可以平复我焦躁的情绪。我会继续尝试卸载，并用基本最近开始的新书尝试填补空白。但书的养分吸收起来太慢了，不及小红书所营造的那种瞬时的满足。

【杂谈】这个夏天，选一条柏香籽

发表于 2022-08-07 更新于 2022-08-08 分类于 杂谈 阅读次数：

最近关注了很多柏香籽的商家，也在各方渠道读了很多关于柏香籽的文章，其实所谓各方渠道，主要是小红书，Youtube上只有寥寥几篇文玩博主的综述性的视频，Google上也搜不到什么，显然这东西还没有渗透进简中之外的世界。我没有安装抖音和B站，想必那里的信息源和小红书也差不多。
小红书真可谓种草神器，只要你想关注的话题，上面都有若干篇质量还算不错的文章帮你测评，试玩，总结，避雷… 真是一刷就停不下来。里面的若干篇文章也煞有介事的帮你厘清当今市面上柏香籽商家的乱象，并”真诚“的推荐了几个工作室。
比较两个极端的例子，其中最大的一家店「WC」基本上打通了从上游供货到下游营销的一整套商业链，淘宝店，直播都做的有声有色，款式充足，全品类覆盖，小几百到大几千的籽都能买得到，推广做的也很到位，甚至在网上发起所谓的「千图计划」，通过顾客回图，既宣示了自己的品牌影响力，也给潜在客户提供了很多设计灵感，大大的种了一波草。而除了寥寥即家头部的大卖家，我所了解到的更多的是小工作室，他们或是从批发商处拿籽，或是小规模从本地收籽，联系加工厂或自己手工加工，然后通过小红书，微信等平台小规模贩售。

【周记】30 七月再见

发表于 2022-07-31 更新于 2022-08-07 分类于 周记 阅读次数：

燥热的七月，这么快就过去了。
突如其来的一场病把这个七月冲散，坚持了半年的运动暂停了一个多礼拜，取而代之的是一些坏习惯，诸如刷直播和小红书种草。
这个礼拜我的生活基本恢复正常，保洁阿姨周二来把家里打扫的焕然一新，周一和朋友出去骑了一圈车，周三打了一场球，体能也基本恢复如初。JY由于咳嗽把肋骨咳坏了，从24号开始过了一个礼拜也没有好转，去看医生不排除有轻微骨裂的可能，只能靠卧床静养，也算新型新冠后遗症。
这个礼拜狗子也不太平，大概是因为乱吃了什么东西，拉了好几天肚子，我们先是给他断粮，吃鸡肉米饭，后来看到便中带血，吓得赶紧送医院检查，化验没问题，X光也没问题，就开了些消炎药在家继续轻食静养。
最后聊聊柏香籽。上一篇结尾我说道从一家定了一串柏香药片，这周做好了，但由于药片厚度比较小，需要150+颗才够挂脖，而这样就超过了原本的预算，同时随着这些天的”调研“，也看到几家口碑不错的商家提供一些品质相当的平价之选，便舍弃了这串小药片，从另一家入了一串厚切药片。同时脑袋一热又从另一处”高冷“的卖家哪里秒杀到了一条”便宜货“藏式斜切猪大肠，最后又从小红书上刷到一个业余玩家自己在热振寺附近自捡籽做的一条大肠手持，有了这三条串，足够自己玩上一段时间了，便决定至此收手，哪成想又陷入了配珠黑洞，苦苦的在网海为这两串珠子（两串大肠打算合并在一起）又设计搭配了两套装饰，终于赶在八月来临之前毕业。
接下来的八月，我要收收心了，多读书，多工作，少刷手机，多盘串。

【周记】29 大病初愈

发表于 2022-07-24 分类于 周记 阅读次数：

这周顺利从Covid毕业，十天转阴。基本从第二周开始就没有太多症状了。JY也转阴了，但还是有些咳嗽，咳得肋骨生疼，网上说有些人的咳嗽症状还是要持续一小段时间，要保持清淡饮食，多睡觉休息。

这周收到了一批海淘的珠子，开心的把自己的星月素串拾掇起来。这周收到JY的影响，也开始关注淘宝直播带货。感觉来到美国之后，很多方面都渐渐和国内脱节，前几天细数起来，有好几项国内火爆的新鲜玩意我都没有尝试过，直播带货算一个，还有抖音（听说抖音的播放量已经超过了Youtube）、游戏方面的吃鸡和王者荣耀，理财方面的花呗借呗，还有老百姓日常都要出示的行程码，想象一下下次回国的体验怕是要很新鲜。如今的直播带货和早年的电视购物最大的不同是，直播带货是双向的，那个屏幕中的卖家可以实时和屏幕前的你交互，回答问题，或者给你推销产品。这种感觉一定程度上打破了传统网购那种浏览+调研式的购物模式，你的安静而隐秘的购物活动，从踏进直播间的第一秒，就会被一句”欢迎大恒“瞬间瓦解，说不上来是好还是坏，更多的是一种新鲜的体验，介于网购和线下购物之间的感觉，进可和主播大聊特聊，退可轻轻一划，遁形于茫茫网海。
JY最近沉迷各种玉器和蜜蜡，我则开始研究最近火起来的柏香籽。JY对这种年年都会生长，存量并不稀有，价格却又被炒高的树种子嗤之以鼻，表示不理解，不欣赏，不反对。

这周泳池的水质在前段时间的super shock之后得到了很大的改善，这周连着三天下水游泳，大概再游个997天就能回本了🙄同时周末还打了两场羽毛球，感觉大病初愈体能上还是有很大的下降，最为明显的是周末和一哥起了一圈车，爬完一个8%大坡之后，我累得体力不支，休息了好半天，还是不能着急，要慢慢回复体能，同时注意饮食，把生病期间瘦的一公斤体重保持住。没错，在生病期间我达到了最近七年体重的最低值，也是前一段时间设定的目标体重，虽然只出现过一天，便反弹回去了一些，但还是要再接再厉，争取更大突破。

【周记】28 群体免疫

发表于 2022-07-17 更新于 2022-07-18 分类于 周记 阅读次数：

抗议三年，一朝破功。我和JY同步感染Covid，这篇开头总结一份我们俩的时间线：
7月9号：密接，和小阳人无保护玩耍6个小时，危险动作包括同桌吃饭，公用一个话筒唱歌，朋友席上已经察觉出身体不适，晚上即测出阳性。 10号：身体没有什么感觉，还出去骑了一圈车。
11号：星期一，自测阴性，我没什么感觉，JY咳嗽，耳鸣，低烧，无精打采，下午测PCR，我搬到
了客厅睡觉。

12号：「阳性第一天」早上起来两人都自测弱阳，昨天的PCR我阴，JY是阳。两个人的症状不太一致，我的症状不温不火，低烧37度，乏力，嗓子略有不适，JY高烧39度，浑身疼，胸闷，嗓子痛，在床上躺了一天，我中午面完试下午便开始请假休息。 13号：强阳，我开始高烧39度，半夜冷醒，泰诺，Advil，阿司匹林换着吃。
14号：开始嗓子疼，继续38度左右的烧。
15号：JY结束高烧，37度左右低烧，我的症状基本开始和她同步
16号：不烧了，嗓子疼，鼻子不通气，两人同事失去嗅觉，味觉还在。
17号：不烧，精力恢复了很多，嗓子疼，轻微咳嗽，检测呈一般阳性（T红条和C红条同时出现），嗅觉继续丧失。

根据网友提供的数据参考，一般自测转阴发生在第十天前后，感染我们的朋友大概是第七天转阴的，顺利的话本周周中我们俩也可以转阴。
然而转阴之后还会继续咳嗽一段时间，而嗅觉的恢复需要的时间更久一些（大概几周）。我现在用四种口味的精油每天做嗅觉训练（Olfactory Training），希望能够帮助神经系统更快拾获这个弄丢了的信号。

这礼拜在抗病毒之余，刷了两部剧，一个是奈飞的关于自闭症律师题材的韩剧「非常律师禹英禑」，另一个是Apple TV+架空历史硬科幻「For All Mankind」。两部都挺精彩的，尤其是FAM，这是我第一部追上的Apple Orginal，之前尝试的几部包括Ted Lasso都没有追到底。有时间的话可以围绕这部剧更深入的谈谈。

【周记】25 被社交媒体支配中

过去的一周感觉自己的可支配时间被压缩的厉害，也许最近依旧沉迷羽毛球，基因中狩猎的本能让我努力的从身边汲取各种羽毛球相关的信息，每个话题或者物件都耗费了我大量的时间来无意识的浏览，搭配个性化算法推荐的那些”有趣“的文章和视频，犹如泥沼令人无法自拔。
这礼拜的一些关于羽毛球的关键词：
• 首先是种草了Bauerfeind护膝，一款价值不菲但身边口碑很好的护膝，由于右膝盖半月板一直有伤，最近打球又比较频繁，感觉膝盖有些吃不消。亲测护膝质量的确不错，不容易掉而且透气和支撑都比较好。
• 然后又从朋友手里用一把vtzf2lcw换了一把2010年的Arcsaber z Slash Taufik（azth）。ArcZ是继08年弓10之后YY推出了两款特别弓剑中的一把，其中ArcFB主打轻量，ArcZ主打小拍面进攻，陶肥也短暂使用过。这把是3UG4，去底胶之后手感还不错，成色也很新，应该打的机会不多，目前主要还是打3U弓10（BG80）和弓11（BGAS）。
• 跟教练练球。这礼拜捡漏约到了湾区这边一个有名的前国手教练，上了一堂两小时的教练课，课程容量还是蛮大的，三个人一组半块场地一直练，由于之前已经连续打了三天，体力有些跟不上，学到了很多实用的技术和知识，包括平抽挡，高远球和杀球的动作，击球后的脚步等等。希望有机会能够一直跟着教练练球。

【周记】24 太浩湖遛狗

发表于 2022-06-19 更新于 2022-06-26 分类于 周记 阅读次数：

这周有点摆烂，新的sprint的第一周，主要的工作是做一些POC，没有太多代码的任务。于是每天白天悠哉的看看文献教程，晚上打球。（后来证明低效的第一周会给后面带来很多无形的压力）。
本周给咖啡机做了一次清洁，周五出门前又做了一次保洁。

星期五下午约上另外三家哈士奇家长一起出发前往太浩湖周末小聚。雪季后我们俩便没有再去过太浩湖，这礼拜正值湾区气温飙高，看天气预报太浩湖那边之后10度上下，十分期待。
星期五的下午，湾区堵得水泄不通，平时4小时上下的路程，我们开了5个半小时。堵车的罪魁祸首是车祸，不知为啥高峰时期车祸就出现的尤为频繁。
自打从科州回来，感觉自己对开长途的阈值又提高了不少，现在一趟Tahoe两百多mile都不需要停车休息了，一口气就沿着I80开到185出口，然后南下来到湖边小木屋。

这是四条哈哈第一次一起出去旅行，也是八位家长第一次同行。来到小木屋后的四条狗子开心的不得了，一直疯玩到吃过晚饭，就渐渐累了，各自找地方趴着。家长们意犹未尽，就着博文带的蓝方，玩了会一夜狼人，Dixit，我和JY对动脑筋的游戏不是很擅长，不过也和大家玩的很开心。晚上11点，印尼公开赛进入到半决赛，看到王祉怡力克何冰娇，挺进决赛 （但最终不敌小戴，拿下超级1000亚军）。

接下来的两天，我们冒着六月飞雪去湖边hiking，也在暖阳下去湖中划船，Oyuki虽然怕在泳池里游泳，但在船上比较放松，好奇的四处张望，还能在浅水里拉着我们的kayak到处跑。

愉快的一周的最后几个小时，我和JY开着车，带着Oyuki，追着夕阳从太浩湖回到温暖的家。
