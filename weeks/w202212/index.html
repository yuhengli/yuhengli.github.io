【周记】12 长春封城

发表于 2022-03-27 更新于 2022-04-02 分类于 周记 阅读次数： Valine：

这周的周记拖了一个礼拜才动笔，实在惭愧。最近几周感觉每天闲下来的可支配的时间愈发变少，细想起来大概原因如下：

  1. Release最后一周，还有很多悬而未决的任务没有完成，所以每天八小时工作时间基本要拉满。
  2. 早起计划落实的不够好，估计8点之前开始恢复行动能力的日子少于3天，而且起床后，上厕所刷手机，去院子里转一圈看看菜地，去厨房转一圈收拾一下未完成的碗筷或者准备早餐，就大概9点了， 距离10点的早会不足一小时，唯一能做的只有见缝插针的练会儿琴
  3. 晚上增加了一些运动计划
  4. 小红书忘了卸载了

于是每天虽然8点到12点都在床下活动，但可供支配的时间依旧捉襟见肘😂

过去一周关注最多的是日益加剧的吉林疫情，那里生活着除了我爸妈以外的几乎全部家人（爸妈在海南过冬，因为疫情原因暂时推迟了返乡的时间）。大东北的人民向来隐忍而幽默，面对突如其来的大规模核酸检查、封城、消杀，他们都积极配合政府工作，不给国家添乱。这段时间每天早上醒来都会刷一遍家里微信群的消息，得知家里有米有粮，人们靠刷抖音来解闷儿，靠发抖音与外界沟通。其实对于健康富裕的阴性家庭来说，封不封楼只是限制了外出活动，增加了些许精神压力。然而封城同时也让这个城市的少数人苦不堪言，病人无法得到及时的治疗；不会用手机的老人逐渐被“扫码”，“抢菜”这些“智能”的防疫措施边缘化；不经世事的孩童要面临第一次离开家长，独自隔离；感染者家里的宠物要面对“无害化处理”；更不用说那些感染者，他们本应是最该被同情和体谅的受害者，但可以想像在如今政策和舆论的压力下，病痛之余，他们要承受多少额外的冷眼和附加伤害。最近打算针对国内这波疫情的所见所闻，单独写一篇文章。

这礼拜的另一个重要的消息是TV+原创剧「CODA」拿到了第一座奥斯卡小金人，虽然风头被Will Smith彻头彻尾的抢走了。「CODA」同时还是第一部来自流媒体播放平台的奥斯卡最佳影片。

此外，本周还收到了月初定的两把弓十，其中3U Taufik的签名款是来自Legends’ Vision版本，4U来自2021年的复刻版。这礼拜还收了一台拉线机，以后可以自己在车库实现拉线自由了。
