【周记】30

发表于 2021-07-28 分类于 周记 阅读次数： Valine：

很惭愧距离上一次更新过去了整整一年。

去年八月，我利用两份工作之间的Gap创建了这个博客，随即加入AWS，九月领养了一只两个月大的哈士奇宝宝Oyuki， 之后的生活变得忙碌异常…

工作上，前三个月独自完成了一个 Re:Invent 项目，简单来说就是一个实现最基本功能的OIDC provider，为License Manager的ISV用户提供一套API来帮助他们创建临时的Access Token，ISV可以将token分发给使用他们License的用户，从而使其无需注册AWS账号也可以使用部分License Manager API。

进入2021年，又先后主持了几个新区域的上线，若干Campaigns，以及License Manager reporting service的设计实现。

生活上，在新的公寓和Ginny宅家工作一整年，逐渐适应了早晚遛狗，白天工作，周末加班的生活。由于两个人的工作都比较忙，居家办公又很大程度上模糊了生活和工作的边界，有一段时间真的感觉压力挺大的，起早贪黑也忙不完工作，生活上一团糟，还要照顾一只调皮捣蛋的哈士奇，这也让我渐渐萌生换一份工作的想法。

2021初夏，在朋友的推荐下，顺利拿到心仪的Offer，和自己并肩奋斗了一年的Team依依惜别。

Gap的这一周幸运的迎来了东京奥运会，终于可以轻松下来看看比赛，做点自己喜欢的事情，包括重新开始在这里写点东西。其实过去一年我并没有放下写日记，几乎每天都会在DayOne上写点东西，只是没有时间整理出来发到博客上，希望以后能够坚持至少每周更新。

这个夏天我和Ginny还幸运的在湾区添置了一处属于我们自己的小家，预计八月底搬家。这是我来到美国之后的第六次搬家，终于搬到了属于自己的家！日后我也会在这里更新一些布置房子，居家生活的心得和故事。

根据美国目前的疫情和打疫苗的情况，预计十月会开始回公司办公，但鉴于近日Delta势头猛烈，还不知道未来会有什么变数。从2020年五月折腾到现在，经历了两次换工作和搬家，也是挺疲惫的，希望2021下半年一切能够好起来。
