【周记】14 平常的一周

发表于 2022-04-10 更新于 2022-04-15 分类于 周记 阅读次数： Valine：

进入四月，过去的一周略显平淡。每天除了工作，运动，剩下的很大一部分时间用来关注日益魔幻的国内疫情。每天都不断有突破底线的故事发生。相比东北人民的隐忍和自娱自乐，上海人民的呼声明显要高抗许多，也就能在各个平台听到更多的冲突与悲剧。

  1. 不知道怎么出沪，凌晨入杭不睡觉就直奔商圈的杭州来福士母女；
  2. 上海本地电视节目教人宅家尝试五天轻断食；
  3. 浦东曹路一阳性居民被带走时，追出的宠物柯基被大白活活打死
  4. 给听障爸爸送吃的女孩，给了远距离接单的骑手200元打赏费，后被网友指责数额太少，受不了网暴跳楼身亡。

DayOne提醒我，去年的本周，我接种了第一针新冠疫苗。随后又陆续种了两针

这周气温回升，我借此机会把泳池好好清理了一番，周五第一次下水游泳，池水蛮干净的，不过已经很凉，而且也许是因为小池子水波比较大，出来觉得天旋地转，是从未体验过的感觉。

这周末开伙做饭，搞了一锅羊蝎子。周日带Oyuki去和他的孪生弟弟玩，两小只打得不亦乐乎。

进入四月，开始每周两到三次羽毛球的计划，周三和周日打强度稍高的男双，周一陪JY练习基本功，也打打混双。同时也会在闲暇时间恶补一下羽毛球的战术和意识，弥补一下双打中球路的不足。

下周就要开启每周回归一天办公，也标志着美国这边进入了后疫情时代。
